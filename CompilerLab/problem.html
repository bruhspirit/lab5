Постановка задачи

Список с инициализацией в Python представляет собой список, который создается с определенными начальными значениями или элементами. Это можно сделать с помощью квадратных скобок и указания элементов через запятые внутри скобок.

Формат записи:

my_list = [1, 2, 3, 4, 5]

Примеры:

my_list = [1, 2.3, "", "с", "fds"]
